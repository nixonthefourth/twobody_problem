\section{Find Gravitational Acceleration}

Step 1, use the equation:

\begin{equation}
\mathbf{\vec{a}} = -\frac{GM}{|r^3|}\vec{r}
\end{equation}

Step 2, substitute variables for values:
\begin{equation}
\mathbf{\vec{a}} = -\frac{1000}{1.95 * 10^6} * (100i + 75j)
\end{equation}

Step 3, hence why:
\begin{equation}
\mathbf{\vec{a}} = (-0.05i - 0.04j)
\end{equation}

This shows an inward acceleration. Imagine a 2D Cartesian plane, and if we put the point $\vec{a}$ on the plane, we would see it in the third quadrant moving towards the center, hence being attracted towards the object. It's worth pointing out that with just the right amount of velocity the object would eitehr fall in, stay within the orbit, or even escape the gravitational pull. Hence, this calculation provides initial acceleration of the oribitng object on the timestep. This matter will be discussed further in the 'Symplectic Integrators' section.
